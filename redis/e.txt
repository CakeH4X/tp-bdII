El comando GEOADD en Redis trabaja sobre una estructura de datos llamada Sorted Set (ZSet). En un ZSet, cada miembro del conjunto está asociado con un valor de puntuación (score) que permite mantener los elementos en un orden específico. Para el caso de datos geoespaciales, Redis utiliza una codificación llamada geohash para convertir las coordenadas geográficas (latitud y longitud) en valores de puntuación que luego se almacenan en el ZSet.
